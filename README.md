# 🎂 Age Calculator

A simple and interactive Age Calculator web app built using **HTML, CSS, and JavaScript**.  
It allows users to calculate their exact age in years, months, and days based on their birth date.

---

## 🚀 Features

- 📅 Easy date of birth input  
- 🎯 Accurate age calculation (years, months, days)  
- ⚡ Fast and user-friendly interface  
- 🎨 Clean and responsive design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
---

## 📂 Project Structure

age-calculator/
│
├── index.html
├── style.css
└── script.js


---

## ▶️ How to Use

1. Open the project in your browser  
2. Enter your date of birth  
3. Click the "Calculate" button  
4. Your exact age will be displayed 🎉
